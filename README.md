# Sales Data Analytics Project

## Overview
This project performs Exploratory Data Analysis (EDA) on a retail sales dataset
to identify sales trends, top-performing product categories, sub-categories,
and regional performance.

## Dataset
- Rows: 9,800
- Columns: 18
- Key fields: Order Date, Sales, Category, Sub-Category, Region

## Tools & Technologies
- Python
- Pandas
- NumPy
- Matplotlib
- Jupyter Notebook

## Analysis Performed
- Data cleaning and preprocessing
- Handling encoding and mixed date formats
- Feature engineering (Year, Month)
- Category-wise sales analysis
- Year-wise sales trend analysis
- Top sub-category identification
- Region-wise sales performance

## Key Insights
- Technology is the highest revenue-generating category
- Sales show strong growth from 2016 to 2018
- Phones and Chairs are top-performing sub-categories
- West region contributes the highest total sales

## Conclusion
The analysis provides actionable insights into business performance using
data-driven techniques and visualizations.
